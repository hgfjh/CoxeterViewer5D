# External Artifact Manifests

This directory stores small, reviewable records for external research tools.
The artifacts themselves may be generated by SageMath, GAP/KBMAG, CoxIter, or a
published-data transcription checker, but the browser app treats every artifact
as inert JSON.

Use `external-artifact-manifest.example.json` as the shape for a checked-in
manifest. A manifest records:

- the tool that produced or checked each artifact;
- the input example and input hash used by that tool;
- the artifact path and optional SHA-256 hash;
- the command shape and runtime environment;
- the narrow claim made by the artifact;
- the boundary saying what the artifact does not certify.

Validate a manifest without running any external algebra system:

```bash
node scripts/validate_artifact_manifest.mjs scripts/certificates/external-artifact-manifest.example.json
```

The validator checks structure, known tool names, referenced paths, and recorded
artifact hashes. It does not certify Coxeter theory, finite quotients, normal
coordinates, or screenshot truth.
